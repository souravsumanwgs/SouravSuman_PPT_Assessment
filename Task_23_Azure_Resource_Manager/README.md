## 🌐 Implementation Steps

### Step 1: Open Resource Group
- Search for **Resource Groups** in the Azure Portal.  
- Open the Resource Group named **ppt-rg**.

### Step 2: Explore ARM Structure
- View all **resources inside the Resource Group**.  
- Observe the **ARM structure** showing hierarchy and deployment of resources.

### Step 3: Deploy a Simple Resource (Optional)
- If needed, deploy a **simple resource** like:  
  - Storage Account  
  - Virtual Machine (VM)  
- Confirm it appears under the **Resource Group**.

### Step 4: Open Cost Management
- Search for **Cost Management + Billing** in the portal.  
- Open the **Cost Analysis dashboard**.

### Step 5: Analyze Cost
- View **usage and cost summary**.  
- Note: Sandbox accounts may show **limited or zero cost**.
