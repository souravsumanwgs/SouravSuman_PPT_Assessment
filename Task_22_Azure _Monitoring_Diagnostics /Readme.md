 Azure Monitoring & Diagnostics (Easy)

## 📌 Objective
Monitor **Azure resources** using **Azure Monitor** and **Metrics**.  
This task demonstrates basic **monitoring and diagnostics** in Azure.

---

## 🛠️ Services Used
- Microsoft Azure Portal  
- Azure Monitor  
- Metrics and Activity Logs  

---

## 🌐 Implementation Steps

### Step 1: Open Azure Monitor
- Search for **Azure Monitor** in the Azure Portal search bar.  
- Open the **Azure Monitor Dashboard**.

### Step 2: View Metrics
1. Click on **Metrics** from the dashboard menu.  
2. Select a **resource** (e.g., Virtual Machine, Storage Account).  
3. View metrics like:  
   - **CPU usage**  
   - **Disk usage**  
   - **Network usage**  

### Step 3: Check Activity Log
- Navigate to **Activity Log** to see **resource activities** and changes.  

### Step 4: Explore Alerts
- Go to the **Alerts** section to get a **basic understanding** of alerting in Azure.

## 🔍 Key Learnings
- Azure Monitor helps track **performance, availability, and health** of resources.  
- Metrics provide real-time insights into **resource utilization**.  
- Activity Logs track all **operations and changes** in the subscription.  
- Alerts can notify admins of **critical events**.
