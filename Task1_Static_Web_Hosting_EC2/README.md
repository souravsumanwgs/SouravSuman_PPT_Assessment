# Task 1: Static Web Hosting in EC2 (Ubuntu + Nginx)

## Simple Definition
Deploy a basic website on an EC2 Ubuntu server using Nginx web server.

## Objective
To understand EC2 instance creation, SSH connectivity, security group configuration, and web server deployment using Nginx.

## Services Used
- Amazon EC2
- Ubuntu Server
- Security Group
- Nginx Web Server

## Steps Performed

1. Launched an Ubuntu EC2 instance.
2. Configured Security Group to allow HTTP (Port 80).
3. Connected to the EC2 instance using SSH.
4. Updated system packages.
5. Installed Nginx web server.
6. Created a sample HTML file inside /var/www/html/.
7. Restarted Nginx service.
8. Accessed the website using Public IPv4 address in browser.
