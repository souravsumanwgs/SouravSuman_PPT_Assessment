TASK 1 – Static Website Hosting on EC2 (Ubuntu + Nginx)

Step 1: Update system packages
sudo apt update

Step 2: Install Nginx
sudo apt install nginx -y

Step 3: Enable Nginx service
sudo systemctl enable nginx

Step 4: Download website template
wget https://www.tooplate.com/zip-templates/2084_zipper.zip

Step 5: Install unzip (if not installed)
sudo apt install unzip -y

Step 6: Extract zip file
unzip 2084_zipper.zip

Step 7: Copy website files to Nginx directory
sudo cp -r 2084_zipper/* /var/www/html/

Step 8: Restart Nginx
sudo systemctl restart nginx

Website is now accessible using Public IP address.
